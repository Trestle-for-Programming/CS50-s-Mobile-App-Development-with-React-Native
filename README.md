# CS50-s-Mobile-App-Development-with-React-Native
CS50’s Mobile App Development with React Native (commonly referred to as CS50M) is a Harvard University course that picks up where CS50x (Introduction to Computer Science) leaves off, transitioning from web development to mobile app development with React Native.

| Week | Topic Area | Key Concepts Covered |
|------|------------|----------------------|
| 1 | **Overview, JavaScript** | History of JS, ES5 vs ES6, variables (`let`, `const`), functions, objects, arrays, scope, hoisting, closures |
| 2 | **JavaScript, ES6** | Arrow functions, template literals, destructuring, spread/rest operators, modules (`import`/`export`), promises, classes |
| 3 | **React, Props, State** | React components (functional/class), JSX, props (read-only), state (`useState`), event handling, re-rendering basics |
| 4 | **React Native** | React Native setup, core components (`View`, `Text`, `Button`, `ScrollView`), styling with `StyleSheet`, platform-specific code |
| 5 | **Lists, User Input** | Rendering lists with `map()`, keys, `FlatList`/`SectionList` in React Native, `TextInput`, handling forms, controlled components |
| 6 | **User Input, Debugging** | Form validation, `useRef`, debugging tools (React DevTools, React Native Debugger, console logs, breakpoints), error boundaries |
| 7 | **Navigation** | React Navigation stack/bottom tabs/drawer, `navigation.navigate`, passing parameters, deep linking |
| 8 | **Data** | Fetching data (`fetch`, `axios`), `useEffect`, loading states, error handling, storing data (`AsyncStorage`, secure storage) |
| 9 | **Expo Components** | Expo overview, managed vs bare workflow, Expo SDK (camera, location, notifications, image picker), permissions |
| 10 | **Redux** | Redux principles (store, actions, reducers), `useSelector`, `useDispatch`, combining reducers, middleware basics |
| 11 | **Async Redux, Tools** | Redux Thunk/Saga, async actions, Redux DevTools, Redux Toolkit (`createSlice`, `configureStore`), persisting state |
| 12 | **Performance** | Memoization (`React.memo`, `useMemo`, `useCallback`), optimizing re-renders, `FlatList` optimizations, bundle size, lazy loading |
| 13 | **Deploying, Testing** | Build for iOS/Android (EAS Build), app store submission, testing (Jest, React Native Testing Library, Detox for E2E) |
